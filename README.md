This repository documents Power BI KPI definitions, DAX measures, and dashboard design logic used to support operational and analytical reporting.
The focus is on how KPIs are calculated, why they matter, and how they are modeled, rather than sharing raw .pbix files.

What This Repository Contains
DAX Measures
A collection of reusable DAX expressions designed for:

KPI calculations
Time intelligence (MTD, YTD, rolling averages)
Submission rates and completion metrics
Conditional logic and data validation

Each measure includes:

Business definition
Calculation logic
Usage context


KPI Definitions
Clear explanations of:

What each KPI measures
Why it is important
How it supports operational or leadership reporting
Common assumptions and edge cases

This ensures metrics are consistent, interpretable, and trusted.

Dashboard Design Notes
Documentation on:

KPI layout decisions
Visual selection rationale
Slicers and drill‑down strategies
Executive vs operational reporting considerations

The goal is to demonstrate analytics storytelling, not just visuals.

Skills Demonstrated

Power BI data modeling
DAX measure creation
Time‑based analysis (YTD, trends)
KPI standardization
Analytical thinking and metric design
Translating business questions into measurable insights


Tools & Technologies

Power BI
DAX
Excel (data preparation)
Power Query (M) for normalization and transformations
